# Как сделать коммит и отправить на GitHub (подробно)

Коммит — это «сохранение» изменений в коде с подписью. После коммита ты отправляешь эти изменения на GitHub командой **push**. Railway потом сам подхватывает новый код и перезапускает бота.

---

## Где писать команды

В **Cursor** внизу открой вкладку **Terminal** (терминал).  
Появится строка вида:

```
PS C:\Users\boiar\tower_bot>
```

Пишешь команды **в эту строку**, по одной. После каждой команды нажимаешь **Enter**.

Если в начале пути нет `tower_bot`, сначала введи и нажми Enter:

```
cd c:\Users\boiar\tower_bot
```

Дальше — три команды по порядку.

---

## Команда 1: добавить файл в «корзину»

Напечатай **ровно** это (можно скопировать):

```
git add bot.py
```

Нажми **Enter**.

**Что делает:** говорит Git: «файл bot.py я хочу включить в следующий коммит».  
Обычно в ответ ничего особенного не пишется — просто появится новая строка с `PS C:\Users\boiar\tower_bot>`.

---

## Команда 2: сделать коммит (сохранить с подписью)

Напечатай **ровно** это:

```
git commit -m "Fix SyntaxError: убрал global в main"
```

Нажми **Enter**.

**Что делает:** создаёт «снимок» текущих изменений и подписывает его этой фразой.  
В ответ должно быть что-то вроде:

```
[main xxxxx] Fix SyntaxError: убрал global в main
 1 file changed, 1 insertion(+), 1 deletion(-)
```

Значит коммит создан.

---

## Команда 3: отправить на GitHub (push)

Напечатай **ровно** это:

```
git push origin main
```

Нажми **Enter**.

**Что делает:** отправляет твой последний коммит на GitHub в репозиторий **tower31**.  
В ответ будет что-то вроде:

```
Enumerating objects: ...
Writing objects: 100% ...
To https://github.com/maksimboyarkin468-ui/tower31.git
   xxxxx..yyyyy  main -> main
```

Значит код ушёл на GitHub. Railway через 1–2 минуты сам подхватит обновление и перезапустит бота.

Если попросит **логин и пароль** — введи логин GitHub и в поле пароля вставь **токен** (не пароль от аккаунта).

---

## Кратко: что вводить и в каком порядке

| № | Что ввести | Enter |
|---|------------|--------|
| 1 | `git add bot.py` | да |
| 2 | `git commit -m "Fix SyntaxError: убрал global в main"` | да |
| 3 | `git push origin main` | да |

Всё вводится в **терминал в Cursor** (вкладка Terminal внизу), **по одной строке**, после каждой — **Enter**.
