# Где посмотреть или создать домен в Railway

Домен нужен, чтобы бот получал обновления из Telegram (вебхук). Без него бот не будет отвечать.

---

## Вариант 1. Settings → Networking

1. **Railway** → открой **проект** → выбери **сервис бота** (карточка с репо).
2. Открой вкладку **Settings** (Настройки).
3. Прокрути до блока **Networking** или **Public Networking** (Публичная сеть).
4. Там может быть:
   - **Ссылка/URL** твоего сервиса (например `xxx.up.railway.app`) — скопируй её.
   - Кнопка **Generate Domain** / **Add Domain** — если домена ещё нет, нажми её; Railway создаст домен и покажет его.
5. Домен вида `название.up.railway.app` — это и есть твой адрес. Вебхук будет: `https://этот-домен/webhook`.

---

## Вариант 2. Карточка сервиса

1. На **главной странице проекта** (где карточки сервисов) кликни по **сервису бота**.
2. Иногда на карточке или в меню есть **открыть приложение** (иконка внешней ссылки или **Open**).
3. Нажми — откроется браузер с адресом твоего сервиса. В **адресной строке** будет домен, например `https://web-production-391a8.up.railway.app/...`.
4. Скопируй домен (без пути), например: `web-production-391a8.up.railway.app`.

---

## Вариант 3. Deployments

1. В сервисе бота открой вкладку **Deployments** (Деплои).
2. Выбери **последний деплой** (успешный).
3. В карточке деплоя иногда есть **URL** или кнопка **View** / **Open** — по ней откроется приложение, в браузере будет виден домен.

---

## После того как узнал домен

1. В **Variables** добавь переменную:
   - **Name:** `WEBHOOK_URL`
   - **Value:** `https://ТВОЙ_ДОМЕН.up.railway.app/webhook`  
     (подставь свой домен, например `https://web-production-391a8.up.railway.app/webhook`).
2. Сохрани переменные — Railway перезапустит сервис, бот сам установит вебхук.

**Либо** один раз открой в браузере (подставь свой домен):

```
https://ТВОЙ_ДОМЕН.up.railway.app/set_webhook
```

Страница вернёт JSON — вебхук будет установлен. Дальше бот начнёт получать сообщения.

---

## Если интерфейс другой

Railway иногда меняет интерфейс. Ищи:

- **Settings** сервиса → раздел про сеть, домены, публичный URL.
- Кнопки **Generate Domain**, **Add Domain**, **Public URL**.
- Ссылку «открыть сервис» / **Open** — по ней в браузере будет виден домен.

Официальная документация: [Public Networking](https://docs.railway.app/guides/public-networking).
